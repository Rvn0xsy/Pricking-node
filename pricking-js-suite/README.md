# Pricking Js Suite

## 模块说明

- modules/cookie.js 获取网页Cookie并打印在控制台上
- ...

## 引入方式

在static.js中添加：

```js
import * as <ModuleName> from './modules/<ModuleName>.js'
```

例如 `cookie.js`：

```js
import * as Cookie from './modules/cookie.js'
```
